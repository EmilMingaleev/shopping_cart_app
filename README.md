# Корзина покупок

Это тестовое задание на Ruby on Rails для управления корзиной покупок


## Установка

1. Клонируйте репозиторий:

    ```bash
    git clone <url>
    cd <папка>
    ```

2. Установите зависимости:

    ```bash
    bundle install
    ```

3. Создайте и мигрируйте базу данных:

    ```bash
    rails db:create
    rails db:migrate
    ```
4. Заполните базу данных тестовыми данными:

    ```bash
    rails db:seed
    ```

4. Запустите сервер:

    ```bash
    bin/dev
    ```
